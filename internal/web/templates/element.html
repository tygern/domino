Properties

The length \(\ell(w)\) of an element is the minimum number of generators needed to express it, equivalently the number of inversions in the signed permutation. The right descent set is \(\{s_i : \ell(w s_i) < \ell(w)\}\) and the left descent set is \(\{s_i : \ell(s_i w) < \ell(w)\}\).

Bad element status

{{if .IsBad}}

This element is bad. It is not a commuting product of generators, yet both \(w\) and \(w^{-1}\) satisfy the right-bad condition. The leading coefficient \(\mu(e,w)\) of the Kazhdan–Lusztig polynomial \(P_{e,w}\) requires special analysis beyond the standard recursive formula.

{{else}}

This element is not bad. Either it is a commuting product of generators or it fails the right-bad condition for \(w\) or \(w^{-1}\).

{{end}}

Domino Tableaux

The right and left domino tableaux are constructed via the Garfinkle insertion algorithm. Each position \(i\) in the signed permutation produces a domino with label \(|w(i)|\): vertical if \(w(i) < 0\), horizontal if \(w(i) > 0\). The right tableau \(R(w)\) is built from \(w\) directly; the left tableau \(L(w)\) from the inverse \(w^{-1}\).

Heap

The heap is a partially ordered set on the generators of a reduced expression. Two generators are comparable when they do not commute, i.e., they are adjacent in the type \(D\) Dynkin diagram. Generators \(s_1\) and \(s_2\) share the leftmost column, reflecting the branch where both connect to \(s_3\).
