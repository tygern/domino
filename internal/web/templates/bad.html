Bad Elements

An element \(w \in D_n\) is bad if it is not a commuting product of generators but both \(w\) and \(w^{-1}\) satisfy the right-bad condition: for each position \(j\), \(w(j) < w(j+2)\) and \(-w(1) < w(3)\). Bad elements are precisely those for which the leading coefficient of the Kazhdan–Lusztig polynomial \(P_{e,w}\) cannot be computed by the standard recursive formula.

The number of bad elements satisfies the recurrence \(a(n) = a(n{-}1) + a(n{-}2) + [n \text{ even}]\).
